Computational Approach

Methods & Data

All molecular data is generated using high-level quantum chemistry methods with consistent protocols applied uniformly across the dataset.

⚛

DFT Properties

Ground-state molecular properties computed using density functional theory with calibrated exchange-correlation functionals optimized for redox potential prediction.

✦

Excited-State Redox

Excited-state oxidation and reduction potentials derived from time-dependent DFT (TD-DFT) using the Rehm–Weller thermodynamic framework.

◎

3D Geometries

Optimized ground-state (S0) and excited-state (S1, T1) molecular geometries stored in standard XYZ and SDF formats.

⊞

ML-Ready Descriptors

Curated molecular fingerprints, SMILES strings, and graph-based representations for direct use in machine learning pipelines.
